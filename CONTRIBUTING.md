# Contributing
Contributing to this project is done by opening a pull request with your suggested changes, sufficiently arguing for them. The maintainers of this project will comment and approve.

NB:
- I am very particular about language, so don't be surprised if I nitpick your phrasing, grammar, or punctuation.
- This file is not very specific in its rules. It applies to the entire repo that rules and changes may come as the repository becomes larger.
