# Open Source Payday 2 Achievement Guide
This is an open source achievement guide for Payday 2. Anyone can contribute with additions, elaborations, corrections, or any other improvements. The goal is a complete and detailed guide to all achievements in Payday 2, such that any achievement hunter can come here to get up-to-date tips and guides for any achievement.

For guidelines on how to contribute, see [CONTRIBUTING.md](./CONTRIBUTING.md), or see the "Contributing" tab above.
